Serve all Wikipedia from a small server

Gozim is a library and a toolset to serve a full Wikipedia copy from small computers even on your own PC is too big.
Think Internet outages, boats, campgrounds or countries without permanent Internet access, Gozim allows a classroom to work on Wikipedia from anywhere no internet needed.

How does it work ?

Thanks to the Kiwix organisation, they provide a full dump of all Wikipedia entries in different languages in a single file (around 40GB for english).
Gozim is capable of reading that file and serve it via HTTP, even on very small servers (Raspberry Pi).
With a 25$ 64GB USB key and a 35$ Raspberry Pi, you get enough power to serve multiple users.
As it serves regular web pages, it works with any WIFI/HTTP capable clients: OLPC, cheap tablets, smartphones...

Why not using the Kiwix tools ?

Kiwix goals are different, they provide a nice graphical client to browse ZIM files locally, but a file as big as 40GB does not target cheap PCs/tablets.
Gozim is a server side tool providing web pages to cheap clients, install it somewhere and forget it.
It was a way to improve my Go skills too.

Technical details

  • Download non indexed ZIM files (Gozim is using its own index system)
  • Create the optional index gozimindex -path=yourzimfile.zim -indexPath=yourzimfile.idx this gives you a full text search on the article's titles
  • Start the web server as follow gozimhttpd -path=yourzimfile.zim -index=yourzimfile.idx and launch your browser to http://yourip:8080
  • See Gozim Github
